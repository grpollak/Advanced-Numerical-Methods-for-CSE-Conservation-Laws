    \begin{defn}[Riemann Problem]\label{defn:riemann_problem}
        Is an initial value problem of a conservation law with
        picewise initial data with a single discontinuity of the form:\\
        \begin{align}
        u_{t}+\flux \left(u\right)_{x}=0&&
        u_{0}=\begin{cases}
                U_{R}&\text{if }x>0\\
                U_{L}&\text{if }x<0
                \end{cases}
        \end{align}
